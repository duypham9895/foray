### Task 1: Create bookmarklet build script

**Goal**: Minify `bookmarklet/foray.js` into URL-encoded `javascript:...` form

**Input**:
- `bookmarklet/foray.js` source file

**Output**:
- Build script: `scripts/build-bookmarklet.ts`
- Minified bookmarklet file: `public/foray-bookmarklet.js` (for reference)
- URL-encoded bookmarklet embedded in settings page

**Code**:

```typescript
// scripts/build-bookmarklet.ts
import fs from "fs";
import path from "path";
import { minify } from "terser";

async function buildBookmarklet() {
  const sourceFile = path.join(process.cwd(), "bookmarklet", "foray.js");
  const source = fs.readFileSync(sourceFile, "utf-8");

  const minified = await minify(source, {
    mangle: true,
    compress: true,
    output: { beautify: false },
  });

  if (minified.error) throw minified.error;

  const code = minified.code || "";
  const encoded = encodeURIComponent(code);
  const bookmarkletUrl = `javascript:${encoded}`;

  // Validate length
  if (bookmarkletUrl.length > 2000) {
    console.warn(
      `Bookmarklet is ${bookmarkletUrl.length} chars (limit 2000)`
    );
  }

  // Write to .env.local for import in settings page
  const envLine = `NEXT_PUBLIC_BOOKMARKLET_URL=${bookmarkletUrl}`;
  console.log(envLine.substring(0, 50) + "...");

  // Also write a reference file
  fs.writeFileSync(
    path.join(process.cwd(), "public", "foray-bookmarklet.js"),
    code
  );

  console.log("✓ Bookmarklet built");
}

buildBookmarklet().catch(console.error);
```

**Verification**:
- [ ] Script runs without errors: `npx ts-node scripts/build-bookmarklet.ts`
- [ ] Bookmarklet URL ≤2000 chars
- [ ] Add to `package.json` scripts: `"build:bookmarklet": "ts-node scripts/build-bookmarklet.ts"`
