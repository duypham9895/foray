## Tasks

### Task 1: Create bookmarklet source file

**Goal**: Write `bookmarklet/foray.js` that captures current page title, URL, and selected text

**Input**: 
- Bookmarklet directory does not exist yet
- No build config for bookmarklet

**Output**:
- New file: `bookmarklet/foray.js`
- Captures: `document.title`, `window.location.href`, `window.getSelection().toString()`

**Code**:

```javascript
// bookmarklet/foray.js
(function() {
  function capturePageInfo() {
    return {
      title: document.title,
      url: window.location.href,
      selectedText: window.getSelection().toString(),
      timestamp: new Date().toISOString(),
    };
  }

  function sendToAPI(data) {
    const API_URL = "http://localhost:3000/api/capture";
    
    fetch(API_URL, {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify(data),
    })
      .then((res) => {
        if (!res.ok) throw new Error(`API error: ${res.status}`);
        return res.json();
      })
      .then((data) => {
        if (data.redirectUrl) {
          window.open(data.redirectUrl, "_blank");
        }
      })
      .catch((err) => {
        console.error("Foray bookmarklet error:", err);
        alert("Foray bookmarklet failed. Check console for details.");
      });
  }

  // Main execution
  const pageInfo = capturePageInfo();
  console.log("Foray captured:", pageInfo);
  sendToAPI(pageInfo);
})();
```

**Verification**:
- [ ] File exists at `bookmarklet/foray.js`
- [ ] File is valid JS (no syntax errors, can be parsed by node: `node --check bookmarklet/foray.js`)
- [ ] Functions `capturePageInfo` and `sendToAPI` are defined

### Task 3: Create build script for bookmarklet URL encoding

**Goal**: Create a Node.js script that URL-encodes the minified bookmarklet into the `javascript:...` form

**Input**:
- `dist/bookmarklet/foray.min.js` exists (from Task 2)

**Output**:
- New file: `scripts/build-bookmarklet.ts` (or `.js`)
- When run: outputs `dist/bookmarklet.txt` with the `javascript:...` link

**Code**:

```typescript
// scripts/build-bookmarklet.ts
import fs from "fs";
import path from "path";

const minifiedPath = path.join(process.cwd(), "dist/bookmarklet/foray.min.js");
const outputPath = path.join(process.cwd(), "dist/bookmarklet.txt");

if (!fs.existsSync(minifiedPath)) {
  console.error(`Minified bookmarklet not found at ${minifiedPath}`);
  console.error("Run: npx vite build --config vite.config.bookmarklet.ts");
  process.exit(1);
}

const minified = fs.readFileSync(minifiedPath, "utf-8");
const encoded = encodeURIComponent(minified);
const bookmarkletLink = `javascript:${encoded}`;

fs.writeFileSync(outputPath, bookmarkletLink, "utf-8");

console.log(`✓ Bookmarklet created: ${outputPath}`);
console.log(`  Length: ${bookmarkletLink.length} chars`);
console.log(`  (Should be <2KB)`);

if (bookmarkletLink.length > 2048) {
  console.warn("⚠ Bookmarklet is larger than 2KB; some browsers may reject it");
}
```

**Verification**:
- [ ] File exists at `scripts/build-bookmarklet.ts`
- [ ] Script runs without error: `npx ts-node scripts/build-bookmarklet.ts`
- [ ] Output file created: `dist/bookmarklet.txt`
- [ ] Output starts with `javascript:` and contains encoded minified code
- [ ] Output is <2KB
