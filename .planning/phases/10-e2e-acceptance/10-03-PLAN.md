# Plan 05-03: Docker reproducibility + CI integration + acceptance verification

**Phase**: Standard-5 (E2E Tests + Acceptance)

**Depends on**: 05-02  
**Effort**: ~1.5 hours

---

## Tasks

### Task 1: Create Dockerfile.e2e

```dockerfile
FROM mcr.microsoft.com/playwright:v1.40.1-focal
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build
CMD ["npm", "run", "e2e"]
```

### Task 2: Add GitHub Actions workflow

```yaml
# .github/workflows/e2e.yml
name: E2E Tests
on: [push]
jobs:
  e2e:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - run: npm ci
      - run: npm run build
      - run: npm run e2e
      - uses: actions/upload-artifact@v3
        if: failure()
        with:
          name: playwright-report
          path: playwright-report/
```

### Task 3: Acceptance verification checklist

Run and verify all tests pass:

```bash
npm run e2e
```

Success criteria:
- [ ] ≥10 E2E specs pass
- [ ] Suite completes in <2min
- [ ] Docker build + run succeeds
- [ ] All Standard features working (bookmarklet, dashboard, search, tags, undo)
