### Task 2: Optional in-app hint on first visit

Use localStorage to show hint once:

```typescript
const [showHint, setShowHint] = useState(() => !localStorage.getItem("shortcutsHintShown"));

useEffect(() => {
  if (showHint) {
    localStorage.setItem("shortcutsHintShown", "true");
  }
}, [showHint]);

if (showHint) {
  return <Toast>Press ? for keyboard shortcuts</Toast>;
}
```
