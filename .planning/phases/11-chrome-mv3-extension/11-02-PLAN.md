# Plan 01-02: LinkedIn + Greenhouse scrapers

**Phase**: Full-1 (Chrome MV3 Extension)  
**Goal**: Site-specific job data extraction for LinkedIn and Greenhouse  
**Depends on**: 01-01  
**Effort**: ~2 hours

---

## Tasks

### Task 1: Create scraper registry

```typescript
// extension/scrapers/index.ts
interface ScraperResult {
  roleTitle: string;
  company: string;
  location: string;
  jobDescription: string;
  salary?: string;
  confidence: number;
}

export async function scrapeCurrentPage(): Promise<ScraperResult | null> {
  const hostname = window.location.hostname;
  
  if (hostname.includes("linkedin.com")) return linkedInScraper();
  if (hostname.includes("greenhouse.io")) return greenhouseScraper();
  
  return null;
}
```

### Task 2: LinkedIn scraper

```typescript
function linkedInScraper(): ScraperResult {
  const roleTitle = document.querySelector("h2.top-card-layout__title")?.textContent || "";
  const company = document.querySelector(".top-card-layout__company-name")?.textContent || "";
  const location = document.querySelector(".top-card-layout__location")?.textContent || "";
  
  return {
    roleTitle,
    company,
    location,
    jobDescription: document.body.innerText,
    confidence: 0.95,
  };
}
```

### Task 3: Greenhouse scraper

```typescript
function greenhouseScraper(): ScraperResult {
  const roleTitle = document.querySelector("h2")?.textContent || "";
  const company = document.querySelector(".postings__heading")?.textContent || "";
  
  return {
    roleTitle,
    company,
    location: "",
    jobDescription: document.body.innerText,
    confidence: 0.90,
  };
}
```
