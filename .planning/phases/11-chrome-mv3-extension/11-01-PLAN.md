# Plan 01-01: Manifest + popup UI + build pipeline setup

**Phase**: Full-1 (Chrome MV3 Extension)  
**Goal**: Set up extension project structure with manifest, popup, and build pipeline  
**Depends on**: Standard-5 complete  
**Effort**: ~2 hours

---

## Success Criteria

- [ ] `extension/manifest.json` MV3 with LinkedIn, Greenhouse, Lever, Ashby permissions
- [ ] `extension/popup.html` + `popup.ts` with React + TypeScript
- [ ] Vite build config outputs to `extension/dist/` with minification
- [ ] Build step: `npm run build:extension`
- [ ] Manual "Load unpacked" works from `extension/dist/`
- [ ] Settings page shows extension version + install instructions

---

## Tasks

### Task 1: Create manifest.json

```json
{
  "manifest_version": 3,
  "name": "Foray Job Tracker",
  "version": "0.1.0",
  "description": "Capture jobs from LinkedIn, Greenhouse, Lever, Ashby with one click",
  "permissions": ["activeTab", "scripting"],
  "host_permissions": [
    "*://*.linkedin.com/*",
    "*://*.greenhouse.io/*",
    "*://*.lever.co/*",
    "*://*.ashby.com/*"
  ],
  "action": {
    "default_popup": "popup.html",
    "default_title": "Foray"
  },
  "background": {
    "service_worker": "background.ts"
  }
}
```

### Task 2: Create popup UI

```typescript
// extension/popup.tsx
export function Popup() {
  return (
    <div className="w-96 p-4">
      <h1>Foray Job Capture</h1>
      <p id="job-info">Analyzing job page...</p>
      <button onClick={capture} className="btn btn-primary">
        Add to Foray
      </button>
    </div>
  );
}
```

### Task 3: Add Vite config for extension build

```typescript
// vite.config.extension.ts
export default defineConfig({
  build: {
    outDir: "extension/dist",
    emptyOutDir: true,
    minify: "terser",
  },
});
```

### Task 4: Add npm scripts

```json
{
  "build:extension": "vite build -c vite.config.extension.ts",
  "dev:extension": "vite build -c vite.config.extension.ts --watch"
}
```
