**Step 1: Create src/app/api/applications/[id]/documents/route.ts**

Two handlers: POST (upload) and GET (list).

```typescript
import { NextRequest, NextResponse } from 'next/server'
import { requireUser } from '@/core/auth/session'
import { ApplicationId } from '@/core/types/ids'
import { uploadDocument, listDocuments } from '@/features/documents/service'
import { uploadSchema } from '@/features/documents/schema'
import type { DocumentKind } from '@/generated/prisma/client'
```

**POST handler:**
1. `const userResult = await requireUser()` — if isErr(), return NextResponse.json({ error: 'Unauthorized' }, { status: 401 })
2. Extract `id` from `params` (use `const { id } = await params` — Next.js 16 async params pattern)
3. Parse multipart form: `const formData = await req.formData()`
4. Extract: `const file = formData.get('file')` (must be File), `const kind = formData.get('kind')` (string), `const notes = formData.get('notes')` (string, optional)
5. Validate file exists: if !file or !(file instanceof File), return 400
6. Validate kind: `const kindResult = uploadSchema.shape.kind.safeParse(kind)` — if !success, return 400
7. Call `const result = await uploadDocument(userResult.value.id, ApplicationId(id), file, kindResult.value as DocumentKind, typeof notes === 'string' ? notes : undefined)`
8. On ok: return NextResponse.json(result.value, { status: 201 })
9. On err with _tag 'Validation': return NextResponse.json({ error: result.error.issues[0]?.message ?? 'Validation failed' }, { status: 422 })
10. On err with _tag 'NotFound': return NextResponse.json({ error: 'Application not found' }, { status: 404 })
11. On other err: return NextResponse.json({ error: 'Internal server error' }, { status: 500 })

**GET handler:**
1. `requireUser()` — 401 if err
2. Extract `id` from params (await params)
3. Call `listDocuments(userId, ApplicationId(id))`
4. On ok: return NextResponse.json(result.value)
5. On err: return NextResponse.json({ error: 'Internal server error' }, { status: 500 })

**Step 2: Create src/app/api/documents/[id]/route.ts**

Two handlers: GET (download) and DELETE.

```typescript
import { NextRequest, NextResponse } from 'next/server'
import { readFile } from 'node:fs/promises'
import { requireUser } from '@/core/auth/session'
import { getDocument, deleteDocument } from '@/features/documents/service'
```

**GET handler (download):**
1. `requireUser()` — 401 if err
2. Extract `id` from params (await params)
3. Call `getDocument(userId, Number(id))`
4. On err NotFound: return 404
5. Read file: `const buffer = await readFile(document.storagePath)`
6. Return with headers:
   ```typescript
   return new NextResponse(buffer, {
     headers: {
       'Content-Type': document.mimeType,
       'Content-Disposition': `attachment; filename="${document.filename}"`,
       'Content-Length': String(document.sizeBytes),
     },
   })
   ```
7. On file read error (ENOENT): return NextResponse.json({ error: 'File not found on disk' }, { status: 404 })

**DELETE handler:**
1. `requireUser()` — 401 if err
2. Extract `id` from params (await params)
3. Call `deleteDocument(userId, Number(id))`
4. On ok: return new NextResponse(null, { status: 204 })
5. On err NotFound: return NextResponse.json({ error: 'Document not found' }, { status: 404 })
6. On other err: return NextResponse.json({ error: 'Internal server error' }, { status: 500 })
